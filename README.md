# Exam-programming-technologies
Іспит з технологій програмування (3 курс. 1 семестр)

Вивчення автоматизації процесів розробки (CI/CD) та підходу «інфраструктура як код» (IaC) передбачає комплексний аналіз методів безперервної інтеграції та доставки, що реалізуються через інструменти оркестрації на кшталт Jenkins та GitHub Actions, у поєднанні з опануванням декларативного опису ресурсів засобами Terraform, що дозволяє формалізувати розгортання обчислювальних середовищ як інженерну дисципліну, гарантуючи ідентичність конфігурацій, прозорість версійності змін та мінімізацію експлуатаційних ризиків при масштабуванні систем.
